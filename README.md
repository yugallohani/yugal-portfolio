## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- npm, yarn, or pnpm

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yugallohaniwww/3d-portfolio.git
    ```

2. Navigate to the project directory:

    ```bash
    cd 3d-portfolio
    ```

3. Install dependencies:

    ```bash
    pnpm install
    # or
    npm install
    # or
    yarn install
    ```

4. Set up environment variables:

    ```bash
    cp .env.example .env.local
    ```

    Add your environment variables:
    - `RESEND_API_KEY`: For contact form emails
    - `UMAMI_DOMAIN`: For analytics (optional)
    - `UMAMI_SITE_ID`: For analytics (optional)

5. Run the development server:

    ```bash
    pnpm dev
    # or
    npm run dev
    # or
    yarn dev
    ```

6. Open [http://localhost:3000](http://localhost:3000) in your browser to see the magic!
